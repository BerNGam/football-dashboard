# ⚽ Football Dashboard

Dashboard interactif des championnats européens de football.
Projet réalisé pour mettre en pratique React et la visualisation de données.

##  Demo

![Dashboard](src\assets\site.png)
##  Stack technique

- **React** + Vite — interface utilisateur
- **Recharts** — graphiques interactifs
- **CSS-in-JS** — styles inline dynamiques

##  Fonctionnalités

- 📋 Classement des équipes avec forme récente
- ⚽ Top buteurs avec barres de progression
- 📈 Évolution des buts par mois
- 🎯 Comparaison des joueurs par radar chart
- 🏆 4 championnats : Ligue 1, Premier League, La Liga, Bundesliga

##  Installation
```bash
git clone git@github.com:BerNGam/football-dashboard.git
cd football-dashboard
npm install
npm run dev
```
